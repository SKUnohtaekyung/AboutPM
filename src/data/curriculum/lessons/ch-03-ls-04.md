## 1. 비즈니스와 수익의 위계

PM은 '미니 CEO'로서 단순히 기능을 기획하는 것을 넘어, 제품이 어떻게 돈을 벌고 생존할지 그 위계를 이해해야 합니다.

#### 비즈니스 위계 4단계
1. **비즈니스 전략 (Strategy)**: "우리는 어떤 시장에서 어떤 가치로 이길 것인가?" (예: 저가 경쟁 vs 프리미엄)
2. **비즈니스 모델 (BM)**: "가치를 어떻게 창출하고 전달하며, 그 대가를 어떻게 확보할 것인가?" (BMC/린 캔버스)
3. **수익화 전략 (Monetization)**: "창출한 가치 중 어느 부분에서, 누구에게 돈을 받을 것인가?"
4. **수익 모델 (Revenue Model)**: "실제로 결제가 일어나는 구체적인 방식은 무엇인가?" (구독, 건당 결제 등)

---

## 2. 가설 검증의 도구: 린 캔버스와 RAT

기존의 BMC(Business Model Canvas)가 운영 단계의 사업을 조망하기 좋다면, 초기 스타트업이나 신규 프로젝트에는 **린 캔버스(Lean Canvas)**가 더 적합합니다.

#### 린 캔버스 vs BMC
- **린 캔버스**: '핵심 문제', '해결 방안', '상위 지표', '경쟁 우위'에 집중합니다. 불확실성이 큰 초기에 "무엇이 틀리면 망하는가?"를 찾는 데 최적화되어 있습니다.
- **RAT (Riskiest Assumption Test)**: MVP를 만들기 전, "가장 위험한 가정(이게 틀리면 사업이 안 되는 것)"부터 아주 작게 테스트하는 기법입니다.

## 3. 5대 수익화 모델 분석

우리 서비스의 성격에 맞는 옷을 골라야 합니다.

#### 0) 주요 수익 모델 카드
- **구독 (Subscription)**: 정기적인 매출(MRR)로 가동률이 높지만, 지속적인 가치 업데이트가 필수입니다. (예: 넷플릭스)
- **프리미엄 (Freemium)**: 기본은 무료, 고급 기능은 유료. 사용자 규모 확보에 유리합니다. (예: 노션, 슬랙)
- **마켓플레이스 (Commission)**: 구매자와 판매자를 연결하고 수수료를 받습니다. 네트워크 효과가 핵심입니다. (예: 배민, 에어비앤비)
- **광고 (Advertising)**: 사용자의 '주의력'을 판매합니다. 아주 큰 트래픽이 필요합니다. (예: 인스타그램, 유튜브)
- **사용량 기반 (Usage-based)**: 쓴 만큼 냅니다. 진입 장벽은 낮지만 매출 예측이 어렵습니다. (예: AWS, API 서비스)
